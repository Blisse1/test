-git init (to initialize a local repository)
-git status (to see whats on the staging area, to pick what files should be tracked)
-git add (to add someething to the staging area)
-git commit -m "message" (to add a message to the file)
-git log (to see what have i saved as a copy/save point)
-git add . (to add every file to the staging area)
-git diff <file> (to see the differences between the file before and after changes)
-git checkout <file name we want back> (to roll back to an older version of the file)
